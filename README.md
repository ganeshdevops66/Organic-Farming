# DevOps CI/CD Project

## Tools Used:
- Terraform (Infrastructure as Code)
- AWS EC2
- Jenkins (CI/CD)
- Docker (Containerization)
- GitHub (Version Control)

## Workflow:
1. Terraform creates AWS EC2
2. Developer pushes code to GitHub
3. Jenkins triggers pipeline
4. Docker builds image
5. App deployed on EC2

## Result:
Fully automated CI/CD pipeline with cloud deployment
